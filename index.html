▶ 실제 사이트로 만드는 방법

데모의 소리는 코드로 합성한 것이고, 실제로는 mp3 파일을 폴더에 넣고 카테고리별로 묶으면 됩니다.

1. 폴더 구조

my-sound-site/
├─ index.html
└─ sounds/
   ├─ animal/  cat.mp3, dog.mp3, bird.mp3 …
   ├─ inst/    piano.mp3, bell.mp3 …
   └─ nature/  wind.mp3, wave.mp3 …

2. 카테고리 버튼 → 랜덤 재생

const FILES = {
  animal: ["sounds/animal/cat.mp3", "sounds/animal/dog.mp3"],
  inst:   ["sounds/inst/piano.mp3", "sounds/inst/bell.mp3"]
};
function playCategory(cat){
  const list = FILES[cat] || [];
  const pick = list[Math.floor(Math.random() * list.length)];
  new Audio(pick).play();   // 버튼 클릭 안에서 호출 → 모바일 재생 OK
}

QR은 ?category= 없이 사이트 주소만 담으면 됩니다. 카테고리 선택은 화면의 버튼이 담당해요. 버튼 클릭이 곧 사용자 동작이라, 모바일 자동재생 제한에도 문제가 없습니다.

데모의 소리는 실시간 합성음입니다 · 실제 배포 시 원하는 mp3로 교체하세요
